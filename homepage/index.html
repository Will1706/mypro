Profil Profesional

Halo! Saya Williyanto Adi. Saya bekerja di bagian Quality Control (QC) Department di industri manufaktur kabel listrik. Selama menjalankan tugas, saya menyadari pentingnya utilitas digital yang cepat dan akurat untuk memverifikasi spesifikasi produk, menghitung parameter uji material, dan mengestimasi dimensi.

Kebutuhan tersebut mendorong saya untuk membangun portofolio ini: sebuah pustaka alat bantu teknik berbasis web interaktif. Saya mengintegrasikan keahlian industri saya dengan teknologi web modern untuk menyederhanakan perhitungan harian QC di pabrik.
